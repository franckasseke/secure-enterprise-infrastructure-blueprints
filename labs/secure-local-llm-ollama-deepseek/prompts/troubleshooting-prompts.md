# Troubleshooting Prompts

This file is reserved for reusable troubleshooting prompts.

## Generic diagnostic prompt

```text
Create a step-by-step diagnostic plan for the following Linux lab issue.
Return: symptoms, assumptions, checks, commands, expected results, possible causes, correction and validation proof.
```

## Documentation prompt

```text
Transform these lab notes into a structured troubleshooting report with context, root cause, fix and lessons learned.
```
