# Documentation Prompts

Reusable prompts for producing consultant-style documentation with a local LLM.

Run with:

```bash
ollama run deepseek-r1:7b
```

---

## README structure

```text
Create a professional GitHub README for a Linux infrastructure lab.
Include context, objectives, architecture, prerequisites, commands, validation, troubleshooting, cleanup and lessons learned.
```

---

## Consultant report

```text
Transform these technical notes into a consultant-style report.
Use clear sections: context, issue, analysis, actions performed, validation evidence, residual risks and recommendations.
```

---

## Architecture note

```text
Create an architecture note from these lab details.
Include components, roles, data flow, security assumptions and operational limitations.
```

---

## Lessons learned

```text
Write a lessons learned section for a Linux lab.
Explain what was built, what failed, how it was fixed and what the engineer should remember.
```
