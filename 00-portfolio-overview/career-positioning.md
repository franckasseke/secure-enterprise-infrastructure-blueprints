# Career Positioning

The portfolio supports a progressive positioning toward **Cloud and Infrastructure Security Architect** roles.

The focus is on securing infrastructure layers that are commonly found in enterprise environments:

- Linux systems
- Cloud platforms
- Container and Kubernetes environments
- Infrastructure automation
- Compliance-oriented security controls
- Secure AI use cases connected to infrastructure

I build and document enterprise-style security projects that demonstrate how infrastructure can be hardened, monitored, automated, validated, and improved.

The objective is not to present isolated tutorials. The objective is to show an engineering and consulting mindset through reproducible technical work.
