# 02 - Cloud Security and Compliance

This domain covers security and compliance patterns for Azure, AWS, GCP, and multicloud environments.

## V1 project targets

1. Azure Governance RBAC Policy Lab
2. AWS Secure VPC Bastion Lab
3. GCP IAM Organization Policy Lab

## Final portfolio target

The final cloud security domain should contain about 9 to 12 strong projects.

Suggested distribution:

- Azure: 3 to 4 projects
- AWS: 3 to 4 projects
- GCP: 2 to 3 projects
- Multicloud patterns: 1 to 2 projects

## Focus areas

- Identity and access management
- Network segmentation
- Secrets management
- Logging and monitoring
- Governance and policy
- Compliance mapping
